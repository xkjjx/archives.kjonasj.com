int main ( int argc,
char * argv[] )

Entry point for the load balancer simulation program.

Simulates a load balancer managing streaming and processing requests over a specified number of clock cycles. The simulation dynamically adjusts server counts based on congestion and demand.

Parameters
argcThe number of command-line arguments.
argvThe command-line arguments:
  • [streaming_server_count]: Initial number of streaming servers.
  • [processing_server_count]: Initial number of processing servers.
  • [clock_cycle_count]: Number of clock cycles to simulate.
  • [–verbose] (optional): Enables verbose output for detailed simulation state.
Returns
0 on success, 1 on failure (e.g., insufficient arguments).
